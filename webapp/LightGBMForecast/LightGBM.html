About Dataset

This dashboard is built using the SWAN-SF Dataset .

Multivariate time series (MVTS) data extracted from solar photospheric vector magnetograms in Spaceweather HMI Active Region Patch (SHARP) series. (2019-01-31)

Angryk, R.A., Martens, P.C., Aydin, B., Kempton, D., Mahajan, S.S., Basodi, S., Ahmadzadeh, A., Cai, X., Filali Boubrahimi, S., Hamdi, S.M., Schuh, M.A. and Georgoulis, M.K. (2020). Multivariate time series dataset for space weather data analytics. Scientific Data, 7(1). doi:https://doi.org/10.1038/s41597-020-0548-x.
